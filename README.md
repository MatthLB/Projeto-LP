# Code x Growth — Landing Page

> Perfil híbrido para marcas que querem crescer com precisão: tráfego pago + desenvolvimento frontend orientado à conversão.


## 🚀 Tecnologias

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) — bundler e dev server
- [Tailwind CSS](https://tailwindcss.com/) — estilização utilitária
- [Framer Motion](https://www.framer.com/motion/) — animações
- [Lucide React](https://lucide.dev/) — ícones

---

## 📁 Estrutura do Projeto

```
src/
├── components/       # Seções da página (Nav, Hero, Footer...)
├── shared/           # Componentes reutilizáveis (GlassCard, SectionTitle)
├── constants/        # Conteúdo estático (textos, links, métricas)
├── lib/              # Utilitários (variantes de animação)
└── App.tsx           # Composição principal
```

---

## ⚙️ Como rodar localmente

**Pré-requisitos:** Node.js 18+ e npm

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/code-x-growth.git
cd code-x-growth

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Acesse [http://localhost:5173](http://localhost:5173) no navegador.

---

## 📦 Scripts disponíveis

| Comando | Descrição |
|---|---|
| `npm run dev` | Inicia o servidor de desenvolvimento |
| `npm run build` | Gera o build de produção na pasta `dist/` |
| `npm run preview` | Visualiza o build de produção localmente |
| `npm run lint` | Roda o ESLint no projeto |

---

## 🌐 Deploy

O projeto está configurado para deploy na [Vercel](https://vercel.com). Basta conectar o repositório e o deploy acontece automaticamente a cada `git push` na branch `main`.
